# Bayanundur Admin

Next.js application for Bayanundur Admin Dashboard.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

The application will be available at [http://localhost:3030](http://localhost:3030).

## Available Scripts

- `npm run dev` - Start development server on port 3030
- `npm run build` - Build the application for production
- `npm run start` - Start production server on port 3030
- `npm run lint` - Run ESLint

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
